# Feature Specification: Multi-Language Greeting

**Feature Branch**: `002-multi-language-greeting`  
**Created**: 2025-09-20  
**Status**: Draft

## User Story

A visitor accesses the ESMuseum Map App and sees a welcoming greeting. Estonian is the default language, but Ukrainian and British English speakers can switch languages via a visible language switcher. The selected language preference is remembered for future visits.

## Core Requirements

- **Language Support**: Estonian (default), Ukrainian, British English
- **Language Switching**: Visible language switcher with instant text changes
- **Persistence**: Remember language preference across visits  
- **Auto-detection**: Show Ukrainian if browser locale is Ukrainian
- **Quality**: Maintain responsive design, accessibility, and HTTPS across all languages
